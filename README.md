# Basic Career Recommendation System
The Career Recommendation System is a Python-based project that recommends suitable career paths based on a user's interests, skills, and preferences. The program asks a series of questions, assigns weighted scores to different careers, and suggests the most suitable option.

## Features
Interactive questionnaire
Weighted scoring system
Career recommendations based on user responses
Explanation of recommendation
Demonstrates core Python concepts

String Manipulation

## How It Works
The user answers a set of career-related questions.
Each answer contributes points to one or more career categories.
Scores are calculated based on predefined weights.
The career with the highest score is recommended.
The system explains why the recommendation was made.

## Learning Objective
This project was created to practice Python fundamentals while building a simple rule-based recommendation system inspired by real-world career guidance tools.
